Python Version: Python 3.7 
Node Version: v11.0.0

Steps:
1. Clone the repo
2. Open cmd cd to the repo
3. Run node app.js 
4. Search localhost:3000/node
5. Store text of api in jsontext.txt
6. Run texttojson.py:
	6.1 Will generate pretty.json
		6.1.1 Generate paid.json
		6.1.2 Generate free.json
7. Run jsontohtml.py:
	7.1 Generate paid.html
	7.2 Generate free.html
